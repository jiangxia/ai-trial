✨ 关键创新设计
协议分流器

在庭审引擎.md 设置智能路由提示词：

```
当检测案件类型：
民间借贷/合同纠纷等 → 激活「民事协议」

故意伤害/贪污贿赂等 → 激活「刑事协议」

行政处罚/行政复议等 → 激活「行政协议」
```

跨协议复用技术

在证据链工坊.md 实现通用提示词模板：

```
【证据处理规则】
{{ 当前协议 }}要求：
✓ 民事案件：采用"高度盖然性"标准
✓ 刑事案件：遵循"排除合理怀疑"原则
但始终执行：证据三性审查（真实性/合法性/关联性）
```

动态角色绑定

角色中枢.md 包含协议适配提示词：

```
在{{ 当前协议 }}模式下：
民事：原告律师 → 被告律师

刑事：公诉人 → 辩护人

行政：行政机关代理人 → 相对人代理人

法官始终作为流程监督者
```
